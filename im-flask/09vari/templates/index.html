模板中变量的使用

1. 简单数据类型的渲染

我的年龄： {{ age }}

我的钱包： {{ money }}

我的名字： {{ name }}


2. 用户信息dict

username： {{ user_info.username }}

nickname： {{ user_info.nickname }}

address.city： {{ user_info["address.city"] }}

address.area： {{ user_info["address.area"] }}


3. 元组和列表

3.1 元组

{{ tuple_city[0] }}
{{ tuple_city[1] }}
{{ tuple_city[2] }}
{{ tuple_city[3] }}


3.2 列表

{{ list_city[0] }}
{{ list_city[1] }}
{{ list_city[2] }}
{{ list_city[3] }}

4. 复杂的数据结构

第一个：
用户名： {{ list_user[0].username}}; 地址：{{ list_user[0]["address"]["city"]}}

第二个：
用户名： {{ list_user[1].username}}; 地址：{{ list_user[1]["address"]["city"]}}
